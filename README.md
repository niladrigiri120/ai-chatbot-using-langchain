# Chatterly 🗣️ – AI Chatbot App

Welcome to **Chatterly 🗣️**, an AI-powered chatbot built with **Streamlit** and **LangChain**!  
Chat, brainstorm ideas, get recommendations, or just have fun conversations—all in **real-time**. 

---

## ✨ Features

- 💬 **Real-time conversation** with AI  
- 🧠 **Smart default prompts**: brainstorming, Q&A, recommendations  
- ⚡ **Streaming responses** for smooth interaction  
- 🔧 **Customizable prompts** and AI behavior  
- 🌐 **Lightweight web app** using Streamlit  

---

## 🎨 Default Prompts

Your chatbot comes with 8 ready-to-use prompts:  

1. 🪶 Write a haiku about AI
2. 🎯 Help me set 3 personal goals
3. 🧩 Give me 3 brain-teasing logic puzzles
4. 👔 Practice a job interview with me
5. ✨ Give me a motivational quote
6. 📘 Summarize the book Ikigai
7. 🎁 Recommend some unique gift ideas
8. ✉️ Draft a polite follow-up email
